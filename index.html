GVortex Downloader

A simple, powerful, and open-source bulk video and audio downloader for Windows, built with C and GTK3. GVortex provides an intuitive graphical user interface for the incredibly versatile `yt-dlp` command-line tool.

Get up and running in minutes.

Follow these simple steps to start using GVortex.

1

Download the Application

Download the latest release `.zip` file from the official releases page and extract it to a folder on your computer.

2

Add Dependencies

GVortex is a graphical front-end for `yt-dlp`. You must download `yt-dlp.exe` and `ffmpeg.exe` from their official websites and place them inside the same folder as `GVortex.exe`.

3

Launch GVortex

That's it! Double-click `GVortex.exe` to run the application and start downloading.

For Developers

Want to build from source? Here's everything you need to know.

1. Environment Setup

First, download and install MSYS2 from msys2.org. Then, open the MSYS2 MinGW 64-bit terminal and run the following commands to get your environment ready.

Update the package database:

pacman -Syu

Install build tools:

pacman -S --needed mingw-w64-x86_64-toolchain mingw-w64-x86_64-gtk3 make

2. Compilation

Once your environment is set up, clone the source repository. Navigate to the project's root directory in your MSYS2 terminal and run `make`.

make

This will compile the source and create `GVortex.exe` in the project directory.

3. Creating a Distributable Package

After a successful compilation, run the provided `copy_dlls.bat` script. This script gathers the final executable, assets, and all required system DLLs into a `dist` folder, which can then be zipped and shared with users.
